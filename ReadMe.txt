This set of m-files demonstrates the main results in [1].  

"demo.m" runs the kernel k-means Nystrom approximation.

To run this demo, simply start Matlab and run the demo file.

Acknowledgment:

The work described in the code will be published in:

[1] Li He and Hong Zhang, Kernel K-means Sampling for Nystrom Approximation, to appear in IEEE Transactions on Image Processing.
